# Auth API

An API to authenticate a user

## API Reference

### Login

```http
    POST /api/auth/login
```

### Signup

```http
    POST /api/auth/signup
```

### Logout

```http
    POST /api/auth/logout
```

## Environment Variables

To run this project, you will need to add the following environment variables to your .env file

`MONGO_URI`

`PORT`

`JWT_SECRET`
